# 📊 Intro to Data Science - Lab Notebook

This repository contains a Jupyter Notebook (`SP21_BSCS_0023.ipynb`) developed as a hands-on laboratory demo for an introductory Data Science curriculum. It outlines core tools, languages, and programming models widely used in the field.

---

## 📝 Notebook Summary

The notebook serves as a starter guide mapping the modern data science ecosystem and implements basic calculations in Python.

### Core Content Covered:
1. **Introduction to Data Science Concepts**: A high-level overview of computational notebooks.
2. **Languages of Data Science**: A compilation of popular analytical programming languages (e.g., Python, R, SQL, Julia, Scala).
3. **Core Libraries**: An index of key python libraries for data engineering, visualization, and modeling (e.g., NumPy, Pandas, Matplotlib, Seaborn, Scikit-Learn, PyTorch).
4. **Data Science Tools Table**: A classification table of developer tools (e.g., Jupyter Notebooks, VS Code, Anaconda, RStudio, Apache Spark).
5. **Mathematical & Arithmetic Operations**: Demonstrations of arithmetic computation and unit conversion using Python scripts.

---

## 💻 Python Demonstrations Included

The notebook executes standard programming assignments:
- **Arithmetic Evaluations**: Formulates and executes simple equations like multiplication and addition:
  ```python
  result = (6 * 4) + 6
  print(result) # Output: 30
  ```
- **Unit Conversions**: A functional conversion script dividing minutes by 60 to calculate equivalent hours:
  ```python
  minutes = 60
  hours = minutes / 60
  print(hours) # Output: 1.0
  ```
